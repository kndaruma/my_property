# 🏠 MyProperty - Commands Documentation
คู่มือการใช้งานคำสั่งทั้งหมดของระบบบ้าน (Advanced Property System)
แบ่งหมวดหมู่ตามสิทธิ์การเข้าถึง เพื่อให้ง่ายต่อการตั้งค่าและให้ข้อมูลกับผู้เล่น

---

## 🔴 1. Admin Commands (เฉพาะแอดมิน)
คำสั่งสำหรับผู้ดูแลเซิร์ฟเวอร์เท่านั้น (ต้องมีสิทธิ์ God, Admin หรือ Ace Perms)

* **`/createhouse [จำนวนช่องเฟอร์นิเจอร์]`**
  * **คำอธิบาย:** สร้างบ้านใหม่ที่จุดที่แอดมินยืนอยู่
  * **วิธีใช้:** ยืนหน้าประตูบ้านที่ต้องการสร้าง แล้วพิมพ์ `/createhouse 150` (สร้างบ้านที่วางเฟอร์ได้ 150 ชิ้น) แอดมินที่สร้างจะกลายเป็นเจ้าของบ้านหลังนั้นชั่วคราว
  * **หมายเหตุ:** ระบบจะ Auto-Backup ข้อมูลบ้านหลังใหม่ให้ทันที

* **`/setslots [HouseID] [จำนวนช่อง]`**
  * **คำอธิบาย:** ปรับเปลี่ยนจำนวนช่องเก็บเฟอร์นิเจอร์สูงสุดของบ้าน
  * **วิธีใช้:** `/setslots 1 200` (ปรับบ้าน ID 1 ให้วางเฟอร์ได้ 200 ชิ้น)

* **`/setpdonate [HouseID] [จำนวนวัน]`**
  * **คำอธิบาย:** แจกสถานะ Donate ให้กับบ้าน (เพื่อปลดล็อคคำสั่งพิเศษ เช่น /setptime)
  * **วิธีใช้:** 
    * `/setpdonate 1 30` (ให้สถานะ Donate 30 วัน แก่บ้าน ID 1)
    * `/setpdonate 1 -1` (ให้สถานะ Donate แบบถาวร)
    * `/setpdonate 1 0` (ลบสถานะ Donate ออก)

* **`/deleteproperty [HouseID]`**
  * **คำอธิบาย:** ลบบ้านและเฟอร์นิเจอร์ด้านในทิ้งถาวร
  * **วิธีใช้:** ต้องยืนอยู่ใกล้ๆ วงวาร์ปหน้าบ้าน แล้วพิมพ์ `/deleteproperty 1` จากนั้นจะมีเมนูขึ้นมาให้กดยืนยันอีกครั้ง

## 👑 2. Owner Commands (เฉพาะเจ้าของบ้าน)
คำสั่งสำหรับผู้ที่เป็น "เจ้าของบ้าน" (มีชื่อเป็น Owner ในระบบ)

* **`/sellproperty [ราคา]`**
  * **คำอธิบาย:** ประกาศขายบ้านของตัวเองให้ผู้เล่นคนอื่น
  * **วิธีใช้:** ยืนที่วงวาร์ปหน้าบ้าน (ด้านนอก) แล้วพิมพ์ `/sellproperty 5000000` (ขายบ้านในราคา 5 ล้าน) ผู้เล่นคนอื่นจะเห็นป้ายราคาที่วงวาร์ป

* **`/cancelsellp`**
  * **คำอธิบาย:** ยกเลิกการประกาศขายบ้าน
  * **วิธีใช้:** ยืนที่วงวาร์ปหน้าบ้าน (ด้านนอก) แล้วพิมพ์ `/cancelsellp` เพื่อดึงบ้านกลับมาเป็นส่วนตัว

---

## 🔑 3. Keyholder Commands (เจ้าของบ้าน & ผู้ถือกุญแจ)
คำสั่งสำหรับเจ้าของบ้าน และผู้เล่นที่ได้รับกุญแจ (แอดมินสามารถใช้ได้เช่นกัน)

* **`/manageproperty`** หรือ **`/mp`**
  * **คำอธิบาย:** เปิดเมนูจัดการบ้านหลัก (จัดการประตู, กุญแจ, เฟอร์นิเจอร์ ฯลฯ)
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/buyfurniture`** หรือ **`/bf`**
  * **คำอธิบาย:** เปิดเมนูร้านค้าเฟอร์นิเจอร์
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/searchfurniture [ชื่อ]`** หรือ **`/sf [ชื่อ]`**
  * **คำอธิบาย:** ค้นหาเฟอร์นิเจอร์จากชื่อหรือโมเดล (รองรับคำย่อ)
  * **วิธีใช้:** `/sf bed` ระบบจะเปิดหน้าต่างแสดงเฟอร์นิเจอร์ทุกชิ้นที่มีคำว่า bed
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/setptime [ชั่วโมง] [นาที]`**
  * **คำอธิบาย:** ล็อคเวลาและสภาพอากาศภายในบ้าน (อิสระจากเวลาเซิร์ฟเวอร์)
  * **วิธีใช้:** 
    * `/setptime 18 30` (ตั้งเวลาในบ้านเป็น 18:30 น.)
    * `/setptime reset` (คืนค่าเวลาให้กลับไปตรงกับเซิร์ฟเวอร์)
  * **เงื่อนไข:** ต้องอยู่ในบ้าน และบ้านหลังนั้นต้องมี **"สถานะ Donate"** เท่านั้น

* **`/peditmode`** หรือ **`/pe`**
  * **คำอธิบาย:** เปิดโหมด "บินอิสระ (Freecam)" เพื่อความสะดวกในการแต่งบ้านและวางเฟอร์นิเจอร์ในที่สูง
  * **วิธีใช้:** พิมพ์ `/pe` เพื่อสลับการเปิด/ปิดโหมดบิน ใช้ WASD บังคับทิศทาง
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/setlift`** หรือ **`/sl`**
  * **คำอธิบาย:** เปิดเมนูสร้างและจัดการลิฟต์ (มิติซ้อนมิติ)
  * **วิธีใช้:** พิมพ์ `/sl` เพื่อสร้างชั้นใหม่ หรืออัปเดตจุดวาร์ปของลิฟต์ในแต่ละชั้น

* **`/setexit [ชื่อประตู]`** หรือ **`/se [ชื่อประตู]`**
  * **คำอธิบาย:** กำหนดจุดทางออก "ด้านในบ้าน" 
  * **วิธีใช้:** เดินไปตรงจุดที่ต้องการให้เป็นทางออกในบ้าน แล้วพิมพ์ `/se Main` (สำหรับประตูหลัก)

* **`/setentrance [HouseID] [ชื่อประตู]`**
  * **คำอธิบาย:** กำหนดจุดทางเข้า "ด้านนอกบ้าน"
  * **วิธีใช้:** ออกไปยืนบนแผนที่โลกหลัก (โลกปกติ) ตรงจุดที่ต้องการทำประตู แล้วพิมพ์ `/setentrance 1 Main`

* **`/lockdoor`** หรือ **`/ld`**
  * **คำอธิบาย:** ล็อคประตูบ้าน (คนที่ไม่มีกุญแจจะไม่สามารถเข้าหรือออกได้)
  * **เงื่อนไข:** ต้องยืนอยู่ในวงวาร์ปของประตูนั้นๆ (เข้าใกล้ระยะ 1.5 เมตร)

* **`/unlockdoor`** หรือ **`/ud`**
  * **คำอธิบาย:** ปลดล็อคประตูบ้าน (ใครก็สามารถเดินเข้าออกได้)
  * **เงื่อนไข:** ต้องยืนอยู่ในวงวาร์ปของประตูนั้นๆ

* **`/invitetop`**
  * **คำอธิบาย:** เรียกดูรายชื่อผู้เล่นที่ยืนรออยู่ "หน้าประตูบ้าน" เพื่อกดเชิญเข้ามาด้านในโดยไม่ต้องปลดล็อคประตู
  * **เงื่อนไข:** ต้องยืนอยู่ใกล้ๆ ประตูทางออกด้านในบ้าน

## 🟢 4. General Commands (ผู้เล่นทั่วไป)
คำสั่งที่ผู้เล่นทุกคนในเซิร์ฟเวอร์สามารถใช้งานได้

* **`/buyproperty`**
  * **คำอธิบาย:** ซื้อบ้านที่กำลังเปิดประกาศขายอยู่
  * **วิธีใช้:** ยืนในวงวาร์ปที่มีข้อความ For Sale แล้วพิมพ์คำสั่งนี้ ระบบจะหักเงินในบัญชีธนาคาร

* **`/knockd`**
  * **คำอธิบาย:** เคาะประตูบ้าน (ส่งเสียงเตือนและป๊อปอัปให้คนในบ้านรู้ว่ามีคนมาหา)
  * **วิธีใช้:** ยืนหน้าประตูบ้านมิติแล้วพิมพ์ `/knockd` (รอสักพักเพื่อให้คนข้างในกดรับ)

* **`/checkhouse`**
  * **คำอธิบาย:** ตรวจสอบ ID ของบ้านหลังนั้นๆ
  * **วิธีใช้:** ยืนตรงวงวาร์ปทางเข้าแล้วพิมพ์คำสั่ง ระบบจะแจ้ง ID ของบ้านให้ทราบ

* **`/checkdim`**
  * **คำอธิบาย:** ตรวจสอบมิติ (Routing Bucket) ที่ตัวเองกำลังยืนอยู่ ณ ปัจจุบัน
  * **วิธีใช้:** พิมพ์ `/checkdim` (โลกปกติจะแสดงผลเป็นมิติ 0)

* **`/dleave`**
  * **คำอธิบาย:** บังคับพาดึงตัวเองออกจากมิติส่วนตัว กลับสู่มิติ 0 (โลกปกติ)
  * **วิธีใช้:** ใช้ในกรณีฉุกเฉิน เช่น บัคตกแมพ หรือหลงมิติ

* **`/clearprops`**
  * **คำอธิบาย:** ลบเฟอร์นิเจอร์ที่เกิดอาการบัค (เช่น ไอเทมตกค้าง, โมเดลโหลดผิดพลาด) ในระยะใกล้ตัว
  * **วิธีใช้:** พิมพ์ `/clearprops` ระบบจะเคลียร์ไอเทมรอบๆ ตัวระยะ 5 เมตร
