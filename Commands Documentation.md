# 🏠 MyProperty - Commands Documentation
คู่มือการใช้งานคำสั่งทั้งหมดของระบบบ้าน (Advanced Property System)
แบ่งหมวดหมู่ตามสิทธิ์การเข้าถึง เพื่อให้ง่ายต่อการตั้งค่าและให้ข้อมูลกับผู้เล่น

---

## 🔴 1. Admin Commands (เฉพาะแอดมิน)
คำสั่งสำหรับผู้ดูแลเซิร์ฟเวอร์เท่านั้น (ต้องมีสิทธิ์ God, Admin หรือ Ace Perms)

* **`/createhouse [จำนวนช่องเฟอร์นิเจอร์]`**
  * **คำอธิบาย:** สร้างบ้านใหม่ที่จุดที่แอดมินยืนอยู่ และ ผู้ที่ใช้คำสั่งจะเป็น Owner โดยอัตโนมัติ
  * **วิธีใช้:** ยืนหน้าประตูบ้านที่ต้องการสร้าง แล้วพิมพ์ `/createhouse 150` (สร้างบ้านที่วางเฟอร์นิเจอร์ได้ 150 ชิ้น) แอดมินที่สร้างจะกลายเป็นเจ้าของบ้านหลังนั้นชั่วคราว 
  แต่หากพิมพ์คำสั่งแค่ `/createhouse` ระบบจะสร้างบ้านที่วางเฟอร์นิเจอร์ได้ 100 ชิ้นอัตโนมัต
  * **หมายเหตุ:** ระบบจะ Auto-Backup ข้อมูลบ้านหลังใหม่ให้ทันที

* **`/setslots [HouseID] [จำนวนช่อง]`**
  * **คำอธิบาย:** ปรับเปลี่ยนจำนวนช่องเก็บเฟอร์นิเจอร์สูงสุดของบ้าน
  * **วิธีใช้:** `/setslots 1 200` (ปรับบ้าน ID 1 ให้วางเฟอร์ได้ 200 ชิ้น)

* **`/setpdonate [HouseID] [จำนวนวัน]`**
  * **คำอธิบาย:** แจกสถานะ Donate ให้กับบ้าน (เพื่อปลดล็อคคำสั่งพิเศษ เช่น /setptime)
  * **วิธีใช้:** * `/setpdonate 1 30` (ให้สถานะ Donate 30 วัน แก่บ้าน ID 1)
    * `/setpdonate 1 -1` (ให้สถานะ Donate แบบถาวร)
    * `/setpdonate 1 0` (ลบสถานะ Donate ออก)

* **`/deleteproperty [HouseID]`**
  * **คำอธิบาย:** ลบบ้านและเฟอร์นิเจอร์ด้านในทิ้งถาวร
  * **วิธีใช้:** ต้องยืนอยู่ใกล้ๆวงวาร์ปหน้าบ้าน แล้วพิมพ์ `/deleteproperty 1` จากนั้นจะมีเมนูขึ้นมาให้กดยืนยันอีกครั้ง

## 👑 2. Owner Commands (เฉพาะเจ้าของบ้าน)
คำสั่งสำหรับผู้ที่เป็น "เจ้าของบ้าน" (มีชื่อเป็น Owner ในระบบ)

* **`/sellproperty [ราคา]`**
  * **คำอธิบาย:** ประกาศขายบ้านของตัวเองให้ผู้เล่นคนอื่น
  * **วิธีใช้:** ยืนที่วงวาร์ปหน้าบ้าน (ด้านนอก) แล้วพิมพ์ `/sellproperty 5000000` (ขายบ้านในราคา 5 ล้าน)

* **`/cancelsellp`**
  * **คำอธิบาย:** ยกเลิกการประกาศขายบ้าน

* **`/setprent [HouseID] [ราคาต่อวัน]`**
  * **คำอธิบาย:** ตั้งป้ายประกาศปล่อยเช่าบ้านแบบรายวัน
  * **วิธีใช้:** พิมพ์ `/setprent 1 5000` (ปล่อยเช่าบ้าน ID 1 ในราคา $5,000 ต่อวัน) วงวาร์ปหน้าบ้านจะขึ้นข้อความให้เช่า

## 🔑 3. Keyholder & Renter Commands (เจ้าของ, ผู้เช่า & ผู้ถือกุญแจ)
คำสั่งสำหรับผู้มีสิทธิ์เข้าถึงบ้าน (รวมถึงแอดมิน) *หมายเหตุ: ผู้เช่าบ้าน (Renter) จะได้รับสิทธิ์เท่าเทียมกับผู้ถือกุญแจทุกประการ*

* **`/manageproperty`** หรือ **`/mp`**
  * **คำอธิบาย:** เปิดเมนูจัดการบ้านหลัก (จัดการประตู, กุญแจ, เฟอร์นิเจอร์ ฯลฯ)
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/buyfurniture`** หรือ **`/bf`**
  * **คำอธิบาย:** เปิดเมนูร้านค้าเฟอร์นิเจอร์
  * **เงื่อนไข:** ต้องอยู่ในบ้านมิติส่วนตัวเท่านั้น

* **`/searchfurniture [ชื่อ]`** หรือ **`/sf [ชื่อ]`**
  * **คำอธิบาย:** ค้นหาเฟอร์นิเจอร์จากชื่อหรือโมเดล (รองรับคำย่อ)
  * **วิธีใช้:** `/sf bed` (เปิดหน้าต่างแสดงเฟอร์นิเจอร์ทุกชิ้นที่มีคำว่า bed)

* **`/setptime [ชั่วโมง] [นาที]`**
  * **คำอธิบาย:** ล็อคเวลาและสภาพอากาศภายในบ้าน (อิสระจากเวลาเซิร์ฟเวอร์)
  * **วิธีใช้:** `/setptime 18 30` หรือ `/setptime reset`
  * **เงื่อนไข:** ต้องอยู่ในบ้าน และบ้านหลังนั้นต้องมี **"สถานะ Donate"** เท่านั้น

* **`/peditmode`** หรือ **`/pe`**
  * **คำอธิบาย:** เปิดโหมด "บินอิสระ (Freecam)" เพื่อแต่งบ้านในที่สูง

* **`/setlift`** หรือ **`/sl`**
  * **คำอธิบาย:** เปิดเมนูสร้างและจัดการลิฟต์ (มิติซ้อนมิติ)

* **`/setexit [ชื่อประตู]`** หรือ **`/se [ชื่อประตู]`**
  * **คำอธิบาย:** กำหนดจุดทางออก "ด้านในบ้าน" 

* **`/setentrance [HouseID] [ชื่อประตู]`**
  * **คำอธิบาย:** กำหนดจุดทางเข้า "ด้านนอกบ้าน" บนแผนที่โลกปกติ

* **`/lockdoor`** หรือ **`/ld`** / **`/unlockdoor`** หรือ **`/ud`**
  * **คำอธิบาย:** ล็อค/ปลดล็อคประตูบ้าน 
  * **เงื่อนไข:** ต้องยืนอยู่ในวงวาร์ปของประตูนั้นๆ

* **`/invitetop`**
  * **คำอธิบาย:** ดึงผู้เล่นที่ยืนรออยู่หน้าประตูบ้านเข้ามาด้านในโดยไม่ต้องปลดล็อค

* **`/cancelrentp`**
  * **คำอธิบาย:** ยกเลิกสัญญาเช่าบ้าน
  * **วิธีใช้ (ถ้าเจ้าของเป็นคนพิมพ์):** จะเป็นการ **เตะผู้เช่าออก** และยกเลิกป้ายประกาศเช่า
  * **วิธีใช้ (ถ้าผู้เช่าเป็นคนพิมพ์):** จะเป็นการ **คืนบ้าน** (ย้ายออก) สัญญาจะขาดทันทีและไม่มีการคืนเงิน

## 🟢 4. General Commands (ผู้เล่นทั่วไป)
คำสั่งที่ผู้เล่นทุกคนในเซิร์ฟเวอร์สามารถใช้งานได้

* **`/buyproperty`**
  * **คำอธิบาย:** ซื้อบ้านที่กำลังเปิดประกาศขายอยู่

* **`/rentp [จำนวนวัน]`**
  * **คำอธิบาย:** ทำสัญญาเช่าบ้านที่เปิดประกาศให้เช่า
  * **วิธีใช้:** ยืนในวงวาร์ปที่ขึ้น For Rent แล้วพิมพ์ `/rentp 7` (เช่า 7 วัน) เงินจะถูกหักจากธนาคารไปให้เจ้าของบ้านทันที

* **`/payrentp [จำนวนวัน]`**
  * **คำอธิบาย:** จ่ายเงินต่อสัญญาเช่า (Extend Rent)
  * **วิธีใช้:** พิมพ์ `/payrentp 3` เพื่อจ่ายเงินต่อสัญญาเพิ่มอีก 3 วัน (มีระบบแจ้งเตือนเมื่อสัญญาเหลือ 3 วัน)

* **`/knockd`**
  * **คำอธิบาย:** เคาะประตูบ้าน (รอคนข้างในกดรับเพื่อวาร์ปเข้า)

* **`/checkhouse`**
  * **คำอธิบาย:** ตรวจสอบ ID ของบ้านที่ยืนอยู่

* **`/checkdim`**
  * **คำอธิบาย:** ตรวจสอบมิติ (Routing Bucket) ปัจจุบัน (โลกปกติคือ 0)

* **`/dleave`**
  * **คำอธิบาย:** บังคับพาดึงตัวเองออกจากมิติส่วนตัว กลับสู่มิติ 0 (โลกปกติ) ใช้แก้บัคตกแมพ

* **`/clearprops`**
  * **คำอธิบาย:** ลบเฟอร์นิเจอร์ที่เกิดอาการบัคในระยะ 5 เมตร
